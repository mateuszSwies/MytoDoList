# MyToDoList
Simple app to adding, removing and marking as a done some tasks. You can watch it here: https://mateuszswies.github.io/MytoDoList/

## Technologies

In this project, I used the following technologies:

- JavaScript
- Bootstrap 5.2.2
- SCSS
